## Tech Stack

This project is built entirely with vanilla web technologies — no frameworks or build tools required.

| Layer      | Technology                                  |
|------------|---------------------------------------------|
| Markup     | HTML5 (semantic, accessible)                |
| Styles     | CSS3 (BEM + CSS Custom Properties)          |
| Scripts    | JavaScript ES2020+ (strict mode, no deps)   |
| Icons      | Font Awesome 6.5                            |
| Typography | Google Fonts — Inter & Poppins              |
| Hosting    | GitHub Pages                                |
| Forms      | Formspree (serverless contact form)         |

## Features

- **Scroll-reveal animations** — elements animate in as the user scrolls using IntersectionObserver
- **Typed text effect** — cycling job titles in the hero section
- **Animated skill bars** — progress bars that fill on scroll
- **Responsive navigation** — hamburger menu with smooth transitions on mobile
- **Scroll progress indicator** — thin bar at the top of the page tracking reading progress
- **Contact form** — serverless form submission via Formspree, delivered to Gmail
- **Fully accessible** — semantic HTML, ARIA labels, keyboard-navigable
- **Dark theme** — consistent dark palette with electric blue and violet accents
- **Performance-first** — no JavaScript frameworks, deferred scripts, optimized assets
